# Total Hotkeys

55 commands with 53 keybindings

#### Core **Text **Formatting (6)

- H1-H6 **Headings**: `ctrl+alt+1` through `ctrl+alt+6`
- Bold: `ctrl+b`
- Italic: `ctrl+i`
- Strikethrough: `ctrl+s`

---

#### Lists & Blocks (12)

- Unordered List: `ctrl+l`
- Ordered List: `ctrl+o`
- Task List: `ctrl+j`
- Blockquote: `ctrl+;` (& advanced `ctrl+shift+;`)
- Indent/Unindent: `tab` / `shift+tab` (& advanced `ctrl+shift+i` / `ctrl+shift+o`)
- Toggle Task Check: `ctrl+shift+j`
- Horizontal Rule: `ctrl+shift+h`

#### Code & Links (6)

- Code Block: `ctrl+u`
- Inline Code: `ctrl+g`
- Link: `ctrl+k` (+ toggle input: `alt+enter`)
- Code Block Input Toggle: `alt+enter`
- Code Block Hide Edit: `escape`
- Select All Code Block: `ctrl+a`

#### Block Operations (6)

- Insert Block Before: `ctrl+shift+b`
- Insert Block End: `ctrl+shift+e`
- Move Block Up: `ctrl+shift+u`
- Move Block Down: `ctrl+shift+d`
- Undo: `ctrl+z`
- Redo: `ctrl+y`

#### Table Operations (13)

- Insert Row Above: `ctrl+shift+f`
- Insert Row Below: `ctrl+=`
- Delete Row: `ctrl+-`
- Insert Column Left: `ctrl+shift+g`
- Insert Column Right: `ctrl+shift+=`
- Delete Column: `ctrl+shift+-`
- Align Left: `ctrl+shift+l`
- Align Center: `ctrl+shift+c`
- Align Right: `ctrl+shift+r`
- Move Caret to Input: `alt+enter`
- Move Caret Previous/Next: `shift+tab` / `tab`

#### Other Features (8)

- Emoji: `:`
- Heading Menu: `ctrl+h`
- Fullscreen: `ctrl+'`
- Increase/Decrease Heading: `ctrl+=` / `ctrl+-`
- At User: `@`
- Remove Mistyped: `backspace`
- Image: (no keybinding)
- Empty Block: (no keybinding)
